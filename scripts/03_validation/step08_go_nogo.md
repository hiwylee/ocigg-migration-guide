## 8-2. Go/No-Go 판정 기준

| 판정 | 조건 |
|------|------|
| **Go** | 중요도 [상] 항목 전체 PASS, WARN 항목 원인 분석 완료 및 비즈니스 영향 없음 확인 |
| **Conditional Go** | WARN 항목 존재하나 조치 계획 수립 완료, 책임자 서명 |
| **No-Go** | FAIL 항목 1건 이상 존재 |

### 중요도 [상] 항목 (FAIL 시 무조건 No-Go)

- [ ] ENABLE_GOLDENGATE_REPLICATION = TRUE
- [ ] Extract/Replicat 상태 RUNNING
- [ ] Extract/Replicat LAG < 30초
- [ ] 타겟 INVALID 객체 0건
- [ ] 타겟 UNUSABLE 인덱스 0건
- [ ] 오브젝트 COUNT 소스/타겟 일치
- [ ] 핵심 테이블 Row Count 일치 (1% 이상 차이 없음)
- [ ] 핵심 테이블 데이터 정합성 PASS
